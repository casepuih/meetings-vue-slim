Initializing Vue.js project

Create your working directory

Your project will have many dependencies (around 200MB for now). In order not to exceed your quota, please work on the host hard drive instead of a shared one.

If you are working on your computer, start from point 4.

  1. Go to /mnt/workspaces/private.
  2. Create a directory with your username there, e.g. mkdir josephina.
  3. Enter it, e.g. cd josephina.
  4. Clone the sample project repository git clone https://github.com/fracz/vue-starter efrei-vue
  5. Enter it cd efrei-vue.
  6. Remove my repository data rm -fr .git.
  7. Install NPM dependencies npm install (may take a while).
  8. Start application with npm run serve.
  9. Head over to http://localhost:8080 and see your Vue.js application.

As always, if you want to stop the server in the terminal, hit Ctrl+C.

See what’s inside

See package.json, node_modules contents, public/index.html, src/main.js and src/App.vue. Make sure you understand their purpose.

IF YOU HAVE TIME

Push the code to your own repo

First, create new repo on GitHub, and then execute the following (adjusting the repository URL):

git init
git add -A
git commit -m "Initial Vue.js project"
git remote add origin https://github.com/YOURNAME/vue-spa
git push -u origin master
